    Data Illustrateur

    Create Expressive Data Visualizations without Programming

    Launch App

    version 2.0

    For best experience, use Google Chrome

    Data Illustrateur aims to make visualization creation as easy as drawing an illustration in vector design tools like Figma and Adobe Illustrator.

    Flexible Mark Creation

    Draw, select and manipulate shapes as if you are working in a vector design interface.

    Interpretable Graphics-Data Join

    Join your data with graphics on canvas through repeat, divide, and densify operations.

    Automatic Visual Encoding

    Bind a data column to a visual property, see the results updated immediately on canvas.
